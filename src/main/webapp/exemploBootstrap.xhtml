Por que usar Bootstrap No artigo de lançamento do Bootstrap, Mark Otto, ao explicar as razões para usar Bootstrap, começa afirmando que o coração do Bootstrap nada mais é do que CSS criado com LESS, um pré-processador destinado a gerar folhas de estilos CSS capaz de oferecer muito mais flexibilidade e poder do que as folhas de estilos convencionais, ou não processadas. Posteriormente foram criadas funcionalidades que permitem usar não somente LESS, mas também SASS como pré-processador CSS para o Bootstrap.
